
# Архитектурная документация проекта ScreensTest (Unity)

## Общая структура `Assets`

```
Assets/
├── Art/                # Графические ассеты проекта
│   └── Sprites/        # Спрайты для UI и геймплейных объектов
├── Plugins/            # Сторонние библиотеки
│   └── DOTween/        # Tween-анимации (внешний плагин)
├── Prefabs/            # Префабы игровых объектов
│   ├── Controllers/    # Префабы управляющих компонентов (например, MainManager)
│   ├── Sound/          # Префабы звуковых компонентов и аудиосистемы
│   └── UI/             # UI-префабы
│       ├── Buttons/    # Кнопки интерфейса
│       ├── Panels/     # Панели и окна UI
│       ├── Slots/      # Слоты для предметов, навыков и т.п.
├── Resources/          # Ресурсы, доступные в runtime
│   └── Configs/        # ScriptableObject-конфиги, доступные через Resources.Load
├── Scenes/             # Сцены проекта Unity
├── Scripts/            # Скрипты логики
│   ├── General/
│   │   ├── Configs/        # ScriptableObject-настройки, используемые системой
│   │   ├── Controllers/    # Управляющие классы: MainManager, UIManager и т.п.
│   │   ├── DataHolders/    # Статические или сериализуемые классы-хранилища данных
│   │   ├── Interfaces/     # Интерфейсы для модульности и слабой связности
│   │   ├── SoundSystem/    # Реализация аудиосистемы: проигрывание клипов, фоновой музыки
│   └── UI/
│       ├── Base/           # Базовые классы UI-элементов (например, Panel)
│       ├── Buttons/        # Скрипты для кнопок (логика нажатий и состояний)
│       ├── Panels/         # Поведение и логика UI-панелей
│       └── Slots/          # Компоненты ячеек инвентаря или интерфейса
├── Settings/           # ScriptableObject'ы и настройки Universal Render Pipeline
├── Sound/              # Аудиофайлы проекта
│   ├── Clips/          # Отдельные звуковые эффекты (SFX)
│   └── Musics/         # Фоновые музыкальные треки (BGM)
└── TextMesh Pro/       # TMP ассеты (структура игнорируется)
```
