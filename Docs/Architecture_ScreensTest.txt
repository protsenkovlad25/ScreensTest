
# Архитектурная документация проекта ScreensTest (Unity)

## Общая структура `Assets`

```
Assets/
├── Art/                # Графические ассеты проекта
│   └── Sprites/        # Спрайты для UI и геймплейных объектов
├── Plugins/            # Сторонние библиотеки
│   └── DOTween/        # Tween-анимации (внешний плагин)
├── Prefabs/            # Префабы игровых объектов
│   ├── Controllers/    # Префабы управляющих компонентов
│   ├── Sound/          # Префабы звуковых компонентов и аудиосистемы
│   └── UI/             # UI-префабы
│       ├── Buttons/    # Кнопки интерфейса
│       ├── Panels/     # Панели и окна
│       ├── Slots/      # Слоты панелей
├── Resources/          # Ресурсы, доступные в runtime
│   └── Configs/        # ScriptableObject-конфиги
├── Scenes/             # Сцены проекта
├── Scripts/            # Скрипты
│   ├── General/
│   │   ├── Configs/        # ScriptableObject-скрипты
│   │   ├── Controllers/    # Управляющие классы
│   │   ├── DataHolders/    # Хранилища данных
│   │   ├── Interfaces/     # Интерфейсы
│   │   ├── SoundSystem/    # Реализация аудиосистемы: проигрывание клипов, фоновой музыки
│   └── UI/
│       ├── Buttons/        # Скрипты кнопок
│       ├── Panels/         # Скрипты панелей
│       └── Slots/          # Скрипты слотов панелей
├── Settings/           # Настройки Universal Render Pipeline
├── Sound/              # Аудиофайлы
│   ├── Clips/           # Звуковые эффекты
│   └── Musics/          # Фоновые музыкальные треки
└── TextMesh Pro/       # TMP ассеты
```
